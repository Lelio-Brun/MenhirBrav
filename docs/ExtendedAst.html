Module ExtendedAst

module ExtendedAst: sig .. end

The simplified Abstract Syntax Tree for grammars.

Compared to Ast, two differences:

  1. groups are sort of "flattened", leaving only productions.
  2. Patterns of the standard library of Menhir are explicit.
type spec = rule list

The grammar.

type rule = {
   name : string; (*

The left-hand side of the rule.

*)
   params : string list; (*

The possible list of parameters.

*)
   prods : production list; (*

The right-hand side of the rule.

*)
}

The rules.

type production = actual list

The productions.

type actual =
| Symbol of string * actual list (*

A possibly applied symbol.

*)
| Pattern of pattern (*

A pattern from the standard library.

*)
| Modifier of actual * modifier (*

A "modified" actual.

*)
| Anonymous of production list (*

An anonymous rule.

*)
