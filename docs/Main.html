Module Main (.ml)

module Main: sig .. end
The main driver of the application.

Once parsed, the grammar is scanned (Scan) to get back the symbols then normalized (Normalize) and transformed (Transform) to a simpler form. Finally, after an optional pass of pattern-recognition (Reduce) it is printed (Printers).
