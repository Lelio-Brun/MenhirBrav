Module Main (.ml)

module Main: sig .. end

The main driver of the application.

Once parsed the grammars are concatenated into one.

The resulting grammar is scanned (Scan) to get back the symbols then normalized (Normalize) and transformed (Transform) to a simpler form. Finally, after an optional pass of pattern-recognition (Reduce) it is printed (Printers).

val get : unit ->
       (string * Lexing.lexbuf) list *
       (Common.Symbols.t -> ExtendedAst.spec -> unit) * (unit -> unit)
  • Returns the lexer buffers, a printer chosen from the according passed options and a function to finally close the input and output channels.
val parse : string * Lexing.lexbuf -> Ast.spec
  • Returns the obtained grammars per input file / lexer buffer.
