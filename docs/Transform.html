Module Transform (.ml)

module Transform: sig .. end

Transform a normalized Ast towards an ExtendedAst.

val transform_actual : Common.Symbols.t -> Ast.actual -> ExtendedAst.actual

Transform an actual. Note that the anonymous rules are transformed with the help of Transform.transform_group.

val transform_symbol : Common.Symbols.t -> string -> Ast.actual list -> ExtendedAst.actual

Transform all non defined symbols (see Common.Symbols.is_defined) corresponding to Menhir standard library symbols (list, pair, option, etc.) into patterns, and all other symbols into symbols.

val transform_modifier : Common.Symbols.t -> Ast.actual -> Ast.modifier -> ExtendedAst.actual

Transform a "modified" actual.

val transform_production : Common.Symbols.t -> Ast.actual list -> ExtendedAst.actual list

Transform a production by transforming its actuals.

val transform_group : Common.Symbols.t -> Ast.group -> ExtendedAst.production

Transform a supposed singleton group by returning the transformation of its unique production.

val transform_rule : Common.Symbols.t -> Ast.rule -> ExtendedAst.rule

Transform a rule by transforming its groups into productions.

val transform : Common.Symbols.t -> Ast.rule list -> ExtendedAst.rule list

Transform a grammar by transforming its rules.
